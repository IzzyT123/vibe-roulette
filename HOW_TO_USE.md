## üêõ Troubleshooting

### "Code isn't updating in preview?"
- Check the browser console (F12) for errors
- Make sure the code has `export default function App()`
- Verify preview tab is selected (not AI tab)

### "AI not generating code?"
- Make sure your prompt includes words like "create", "build", "make", "generate"
- Check if Mock AI is selected (Settings ‚Üí Mock AI should be default)

### "Want better AI responses?"
- Configure a real API key (OpenAI or Claude)
- Click Settings (‚öôÔ∏è) ‚Üí Choose provider ‚Üí Add key
- Real AI generates much smarter, production-ready code!
